# Fonts Setup

This project uses the following fonts:

## Founders Grotesk
- **Regular** (font-weight: 400)
- **Medium** (font-weight: 500)
- **Semibold** (font-weight: 600)
- **Bold** (font-weight: 700)

### Where to get Founders Grotesk
Purchase a web font license from [Klim Type Foundry](https://klim.co.nz/retail-fonts/founders-grotesk/)

### Font files needed
Place these WOFF2 files in the `/fonts/` directory:
- `founders-grotesk-regular.woff2`
- `founders-grotesk-medium.woff2`
- `founders-grotesk-semibold.woff2`
- `founders-grotesk-bold.woff2`

## Syne
The ROAM branding uses **Syne** at font-weight 800, loaded from Google Fonts (already included in the HTML).

## Alternative: Use a free font
If you don't want to purchase Founders Grotesk, you can replace it with a free alternative like:
- [Inter](https://fonts.google.com/specimen/Inter)
- [Work Sans](https://fonts.google.com/specimen/Work+Sans)
- [Manrope](https://fonts.google.com/specimen/Manrope)

To switch to a free font, update the `@font-face` declarations in the CSS and the Google Fonts link.
